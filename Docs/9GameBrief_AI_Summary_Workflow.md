# GameBrief --- AI Patch Summary Workflow

## Purpose

Transform long patch notes into short summaries for casual gamers.

## Workflow

1.  Patch notes imported into database
2.  Raw content stored in `patches.raw_content`
3.  AI processes content
4.  Summaries saved in `patch_summaries`

## Summary Types

Quick Summary Casual Impact Should I Log In

## Example

Raw Patch Notes → AI Processing → 3 Summaries → Display on Patch Page

## Future Improvements

-   Auto summarization pipeline
-   Queue processing
-   Multiple AI models
