# GameBrief --- Admin Workflow Guide

## Purpose

Allow admins to manage game data, patches, and events.

## Admin Responsibilities

### Adding Games

Games primarily imported from IGDB.

Admin verifies: - cover image - slug - game name

### Adding Patches

Steps: 1. Find official patch notes 2. Paste raw patch content 3. Add
source URL 4. Save patch

AI summaries generated afterwards.

### Adding Events

Steps: 1. Enter event title 2. Add description 3. Set start and end time
4. Attach to game

Users can then create reminders.
