# GameBrief Demo Script


Start on the `homepage`

"I usually lose track of the games I actually care about, so what I want here is one place to keep on top of updates without checking five different places."

Pause briefly on the landing page.

"This already gives me a feel for the app with the different visual themes and the overall mood, so it doesn't feel flat or generic."

"I can also see trending games straight away, which is useful because even before I search for anything, I'm getting a sense of what's active right now."

"I can go straight into browsing, which is probably what I'd do first anyway."

Go to `Games`.

"First thing I'm doing is finding the games I actually play. I can search, filter, and cut the list down quickly instead of scrolling through everything."

One or two filters `multiplayer` and `free to play `

"This is already useful because I'm only looking at games that are relevant to me."
"I can see a few games here that I play , lets favourite them"

"After a few favourites, the app starts feeling much more tailored instead of generic."

Open a game detail page `Warzone`

"Once I open a game, I can immediately see the stuff I'd actually want to check, like recent updates and upcoming events."

Open a `patch` from one of the game pages.

"This is where I'd probably get the most value, because patch notes are usually long and I normally just want the important part."

Show the chatbot.

"Instead of reading the whole thing, i can ask the chat to summarise it for me. Instead of digging through the patch notes myself."

"That makes it feel a lot more practical, because I can ask follow-up questions instead of starting over every time."

Go to the main `Patches` then to `view all my patches ` page.

"Now that I've favourited a few games, I don't need to jump between pages. I can just come here and see updates in one place."

"And if I want help understanding any of these updates, I can open one up and use the chatbot to summarise the patch for me in a much faster, simpler way."

Go to `Events`.

"I also want to know what's coming up, not just what's already changed. This saves me from missing events for games I care about."

Open an event.

"If something looks important, I can open it, check the details, and save it properly instead of hoping I remember later."

Save a reminder.

"That's where it becomes genuinely useful, because now it's helping me remember things, not just showing me information."

Go to `My Profile`.

"Once I've saved a few things, this starts to feel like a gamers' hub"

"I can also move between the For You and Communities sections here, so it feels like more than just a tracker.
My good friend Echo just posted about Valorant.

Here I can see my friend Echo is part of the FPS community, I feel like id be a good fit for that too! Let me join it.
It starts to feel like a space built around the kinds of games and conversations I want to keep up with."

Open `My Games`.

"This is useful because I don't need to search all over again, I can just jump back into the games I've favourited."

Open `My Patches`.

"This is probably where I'd spend the most time, because it's only showing me updates tied to the games I care about."
`New` badge is visible:
"The freshness cues are useful too, because I can tell what's new at a glance."

Open `My Events`.

"And this makes it easy to check what I've saved without trying to remember dates myself."

"If I want to, I can also find other users from here, so it doesn't feel like a solo tracker only."
